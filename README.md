# 多媒体播放器

**一个基于 Qt6 + FFmpeg + SDL2 的全能多媒体播放器，集成音视频播放、B站网络流、多任务下载、工具箱等完整功能。**

## ✨ 核心特性

### 🎬 播放能力
- **本地播放**：支持 MP4、AVI、MKV、MOV、FLV、WMV 等主流格式
- **网络播放**：B站视频直接播放（yt-dlp + RTMP 推流）
- **HLS 点播**：支持 `.m3u8` 直播/点播流
- **音视频同步**：基于音频主时钟的精准同步
- **硬件加速**：SDL2 渲染，支持 Direct3D

### 用户体验
- **水元素主题**：深度定制的 QSS 主题
- **动态壁纸**：14 图层独立动画（粒子、水纹、光晕）
- **无边框窗口**：自定义标题栏 + 导航栏

### 📥 下载管理
- **多任务队列**：支持同时下载多个文件
- **断点续传**：HTTP Range 支持，可暂停/恢复
- **速度显示**：实时下载速度
- **历史记录**：SQLite 持久化存储

### 🛠 工具箱
- **视频转码**：多种格式互转
- **去水印**：智能区域去水印
- **视频裁剪**：裁剪视频片段

### ⚙️ 技术架构
- **6 线程模型**：解复用、视频解码、音频解码、渲染准备、SDL 音频、主线程
- **无锁队列**：高效线程间数据传递
- **双缓冲渲染**：避免画面撕裂
- **插件化设计**：解码器/渲染器可扩展

## 🚀 快速开始
### 1. 克隆项目

git clone

### 2. 准备第三方库
```bash

3rdparty/
├── ffmpeg/
│   ├── include/  # FFmpeg 头文件
│   └── lib/      # FFmpeg 库文件
├── sdl2/
│   ├── include/  # SDL2 头文件
│   └── lib/      # SDL2 库文件
└── sqlite3/
    ├── include/sqlite3.h
    ├── lib/libsqlite3.dll.a
    └── bin/sqlite3.dll

### 3. 编译
```bash
mkdir build && cd build
cmake .. -G "MinGW Makefiles"
mingw32-make

### 4. 配置运行时环境
将以下文件放入 `bin/` 目录：
- `yt-dlp.exe`
- `cookies.txt`（B站登录凭证，从浏览器导出）
- `SDL2.dll`
- FFmpeg 系列 DLL
- `sqlite3.dll`

### 5. 启动 Nginx（网络播放必需）
```bash
cd C:\nginx-rtmp-win32
start nginx

## 📖 使用指南

### 本地播放
1. 点击「打开文件」
2. 选择视频文件
3. 自动播放

### B站网络播放
1. 点击「网络播放」
2. 输入 B站视频链接
3. 自动解析并播放

### 下载视频
1. 切换到「下载」页面
2. 点击「新建下载」
3. 输入 URL 和保存路径
4. 支持暂停/恢复
